## Getting Started

First, run the development server:

```bash
# Install ddev cf: https://ddev.readthedocs.io/en/stable/
# Start development server
ddev start
```

Open [https://eve-tools.ddev.site](https://eve-tools.ddev.site) with your browser to see the result.
